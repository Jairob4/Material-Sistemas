# Material-Sistemas

Repositorio oficial de apoyo para **monitores académicos de Ingeniería en Sistemas**.

Aquí se centraliza material de estudio y acompañamiento para distintas materias, con una organización por ramas para mantener el contenido claro y separado.

## ¿Cómo está organizado?

El contenido se agrupa por **ramas (branches)**:

- Cada rama representa una **materia diferente**.
- Dentro de cada rama se publica el material correspondiente a esa materia (guías, ejercicios, talleres, recursos, etc.).
- La rama principal se usa como punto de entrada e información general del repositorio.

## Objetivo del repositorio

- Facilitar el acceso al material académico de monitorías.
- Mantener una estructura ordenada por asignatura.
- Permitir que el contenido evolucione de forma independiente por materia.

## Uso recomendado

1. Identifica la materia que necesitas.
2. Cambia a la rama correspondiente.
3. Consulta o descarga el material publicado en esa rama.
